# Uplink CuebSAT: Mission Flight Software 
**Participant**: Rafay Imran
**ProjectID**: Uplink-Rafay Imran

## Mission Overview
This is a 1U CubeSAT project develeped for HackClub program. The goal is to build a reliable flight software stack in python using Raspberry Pi Zero 2W that monitors power, logs 9-DOF motion data, and captures orbital imagery.

## Tech Stack:
- **Language:** Python
- **Libraries:** `time`, `logging`, `picamera` (Updating..)
- **Hardware:** Raspberry Pi Zero 2W, Sony IMX219 Camera, 9-DOF IMU.
